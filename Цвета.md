# Цвета в логах GitLab CI/CD

## Пример 

```bash
echo -e "\e[38;5;196mЯрко-красный (196)\e[0m"
# Жирный красный текст на жёлтом фоне
echo -e "\e[1;31;43mВажное сообщение\e[0m"
```
Обязатольно в конце надо сбросить

## Основные цвета

| Цвет | Текст | Фон |
|------|-------|-----|
| Чёрный | `\e[30m` | `\e[40m` |
| Красный | `\e[31m` | `\e[41m` |
| Зелёный | `\e[32m` | `\e[42m` |
| Жёлтый | `\e[33m` | `\e[43m` |
| Синий | `\e[34m` | `\e[44m` |
| Фиолетовый (Magenta) | `\e[35m` | `\e[45m` |
| Голубой (Cyan) | `\e[36m` | `\e[46m` |
| Серый/Белый | `\e[37m` | `\e[47m` |

## Стили текста

| Стиль | Код | Сброс |
|-------|-----|-------|
| Жирный (Bold) | `\e[1m` | `\e[22m` |
| Приглушённый (Dim) | `\e[2m` | `\e[22m` |
| Курсив | `\e[3m` | `\e[23m` |
| Подчёркнутый | `\e[4m` | `\e[24m` |
| Мигающий | `\e[5m` | `\e[25m` |
| Инвертированный | `\e[7m` | `\e[27m` |
| Скрытый | `\e[8m` | `\e[28m` |

## 256 цветов

```bash
# Формат: \e[38;5;{N}m
echo -e "\e[38;5;196mЯрко-красный (196)\e[0m"
echo -e "\e[38;5;46mЗелёный (46)\e[0m"
echo -e "\e[38;5;208mОранжевый (208)\e[0m"
echo -e "\e[38;5;21mСиний (21)\e[0m"
```


|Действие |	Код |
|-------|-----|
| Сброс всех стилей	 | \e[0m |
| Сброс жирного/приглушённого |	\e[22m |
| Сброс курсива	| \e[23m |
| Сброс подчёркивания	| \e[24m |
